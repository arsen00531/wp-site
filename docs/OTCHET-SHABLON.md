## 1. Цель работы

Освоить интерфейс WordPress и базовые инструменты CMS: создание страниц, работа с блоками, загрузка изображений и публикация сайта.

## 2. Концепция проекта

**Тема:** Фестиваль цифрового искусства «Pixel Horizon 2026».

**Целевая аудитория:** студенты дизайна и разработки, молодые иллюстраторы, инди-геймдев.

**Идея:** одностраничный лендинг вымышленного трёхдневного фестиваля (лекции, воркшопы, конкурс digital-работ, VJ-night).

## 3. Сценарии использования

| Роль | Сценарий |
|------|----------|
| Гость | Заход на главную → Hero → кнопка «Программа» → расписание → «Зарегистрироваться» → CTA |
| Студент | Секция «Работы участников» → вдохновение → FAQ «конкурс» → подача заявки |
| Спикер / пресса | Блок «Контакты» → email / Telegram |

*(Приложите блок-схему: прямоугольники = экраны-секции, стрелки = переходы по якорям #schedule, #cta.)*

## 4. Реализация в WordPress

- Платформа: WordPress (wordpress.com **или** локальный Docker)
- Тема: **Twenty Twenty-Five**
- Тип: одна страница, секции = блоки Gutenberg
- Навигация: якорные ссылки, `scroll-behavior: smooth`

### Использованные блоки

Обложка, Колонки, Медиа и текст, Таблица, Галерея, Детали (FAQ), Цитата, Кнопки, HTML (карта).
