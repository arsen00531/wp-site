# Практические работы №3–4 — Фестиваль цифрового искусства (WordPress)

**ПР №3** — одностраничный лендинг. **ПР №4** — многостраничный сайт (4 страницы + меню) на основе того же проекта.

Дисциплина: «Разработка и прототипирование веб-сайтов и интерфейсов онлайн» (ИПТИП).

## Что в проекте

| Компонент | Назначение |
|-----------|------------|
| `docker-compose.yml` | Локальный полноценный WordPress + MySQL |
| `wp-content/plugins/festival-pr3-landing/` | Плагин: контент в блоках Gutenberg, ПР №4 — 4 страницы и меню |
| `scripts/setup.sh` | Автоустановка WP, тема **Twenty Twenty-Five**, активация лендинга |

### Секции лендинга (как в ТЗ)

1. **Hero** — Cover, кнопки «Зарегистрироваться» / «Программа»
2. **Преимущества** — Колонки (3)
3. **О программе** — Медиа и текст
4. **Спикеры** — Колонки + фото
5. **Расписание** — Таблица (4 строки)
6. **Портфолио** — Галерея (6 изображений)
7. **FAQ** — блок «Детали»
8. **Отзывы** — 3 цитаты
9. **CTA** — Обложка + кнопка «Подать заявку»
10. **Контакты** — текст, список, карта (OpenStreetMap)

### ПР №4 — структура сайта (4 страницы)

| Страница | Содержание |
|----------|------------|
| **Главная** | Hero, преимущества, о фестивале (кратко), превью спикеров, портфолио, FAQ, отзывы, контакты |
| **Программа** | О фестивале, расписание (таблица), дни 2–3 |
| **Спикеры** | Карточки спикеров с описанием выступлений |
| **Регистрация** | Форма заявки и CTA |

Меню в шапке (Главная · Программа · Спикеры · Регистрация), на мобильных — выезжающая панель (бургер).

## Вариант A — локально (Docker) — рекомендуется для отработки

### Требования

- Docker и Docker Compose
- curl

### Запуск

```bash
cd /home/arsen00531/mirea/wordpress
cp .env.example .env   # если ещё нет
./scripts/setup.sh
```

Откройте в браузере:

- **Сайт:** http://localhost:8080  
- **Админка:** http://localhost:8080/wp-admin  
- Логин по умолчанию: `admin` / `admin123` (меняется в `.env`)

### Ручная настройка (если скрипт не запускали)

1. `docker compose up -d`
2. Пройдите мастер установки WP в браузере
3. **Плагины → Festival PR3 Landing → Активировать**
4. **Инструменты → Festival PR3** — проверить ссылку на лендинг

### Редактирование

- **Страницы → «Фестиваль цифрового искусства»** — редактор блоков (как в ТЗ)
- Или правка файла `wp-content/plugins/festival-pr3-landing/content/landing-blocks.html` и кнопка **Пересоздать лендинг** в **Инструменты → Festival PR3**

### Остановка

```bash
docker compose down
```

## Вариант B — WordPress.com (как в задании преподавателя)

1. Перейти на https://wordpress.com/ru  
2. Создать аккаунт → **Попробуйте бесплатную версию**  
3. **Внешний вид → Темы** → активировать **Twenty Twenty-Five** → **Настроить сайт**  
4. **Страницы → Добавить** → название «Фестиваль цифрового искусства»  
5. Добавлять блоки через **+** по структуре из `content/landing-blocks.html` (можно копировать секции по очереди в редактор)  
6. **Настройки → Чтение** → главная страница = эта страница  
7. Плавный скролл: **Оформление → Стили → ⋮ → Дополнительные стили:**

```css
html {
  scroll-behavior: smooth;
}
```

8. Якоря: у блока **Дополнительно → HTML-якорь** (`about`, `schedule`…), ссылки кнопок `#about`, `#schedule` (Ctrl+K).
